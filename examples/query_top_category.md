# User Story #2: «Какая категория обходится мне дороже всего?»

## Ожидаемый ReAct-трейс

```
============================================================
[Thought]
Нужны все обязательства. Группирую по category, для каждого считаю
сумму в RUB. RUB-записи оставляю, не-RUB конвертирую в RUB.
============================================================

============================================================
[Action]
tool=get_obligations
input={"status": null, "category": null}
============================================================

============================================================
[Observation]
[12 записей в 5 категориях: subscription, software, utility, membership, donation]
============================================================

============================================================
[Thought]
Категории и их записи:
- subscription: Netflix (USD 9.99), Spotify (USD 5.99), Яндекс.Плюс (RUB 299),
  iCloud (USD 2.99), Amazon Prime (USD 8.99, paused), The Guardian (GBP 5.99)
- software: GitHub Pro (USD 4.00), JetBrains (EUR 24.90)
- utility: Мосэнерго (RUB 1500), Ростелеком (RUB 800)
- membership: World Class (RUB 4500)
- donation: Patreon (EUR 5.00)

Решаю вопрос: включать ли paused? Спрошу только active — это корректнее.
============================================================

============================================================
[Action]
tool=get_obligations
input={"status": "active", "category": null}
============================================================

============================================================
[Observation]
[11 записей без Amazon Prime]
============================================================

============================================================
[Thought]
Конвертирую все не-RUB суммы в RUB. Будет несколько вызовов
convert_currency, можно использовать кешированные курсы.
============================================================

============================================================
[Action]
tool=convert_currency
input={"amount": 9.99, "from_currency": "USD", "to_currency": "RUB"}
============================================================

============================================================
[Observation]
{"ok": true, "amount": 774.31, "rate": 77.49, "source": "cbr"}
============================================================

... (конвертация всех не-RUB сумм) ...

--- ANSWER ---
Самая затратная категория (только active подписки, в RUB):

| Категория     | Кол-во | Итог в RUB |
|---------------|-------:|-----------:|
| utility       |      2 |  2 300.00  |  (Мосэнерго 1500 + Ростелеком 800)
| software      |      2 |  2 514.26  |  (GitHub Pro 309.96 + JetBrains 2204.30)
| subscription  |      5 |  2 393.00  |  (Netflix 774.31 + Spotify 464.07 + Яндекс.Плюс 299 + iCloud 231.69 + The Guardian 620.97 ≈ 2389.97)
| membership    |      1 |  4 500.00  |  (World Class)
| donation      |      1 |    442.63  |  (Patreon)

**Дороже всего: категория membership — 4500.00 RUB (только World Class).**

Если не учитывать membership как единственную запись, то лидирует software.
```

## Что демонстрирует

1. **Уточнение логики** — агент сам решает, что Amazon Prime на паузе не считаем.
2. **Группировка по категориям** с конвертацией всех валют в RUB.
3. **Бонусная аналитика** — не только лидер, но и распределение.
4. **Кеш курсов** — повторные конвертации используют тот же rate.
