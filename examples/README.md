# Примеры использования агента

В этом каталоге — примеры запросов к агенту и ожидаемых трейсов.

## Содержание

| Файл | Описание |
|---|---|
| `query_30_days_rub.md` | User Story #1 — «Сколько я потрачу в ближайшие 30 дней? Покажи итог в рублях.» |
| `query_top_category.md` | User Story #2 — «Какая категория обходится мне дороже всего?» |
| `query_this_week.md` | User Story #3 — «Есть ли у меня платежи на этой неделе?» |
| `query_error_handling.md` | Антигаллюцинация — поведение при ошибке API |

## Как воспроизвести

```bash
# One-shot режим
python main.py "Сколько я потрачу в ближайшие 30 дней? Покажи итог в рублях."

# Или через Docker
docker compose run --rm agent "Сколько я потрачу в ближайшие 30 дней? Покажи итог в рублях."

# С логом трейса в файл
TRACE_LOG_PATH=./trace.jsonl python main.py "ваша вопрос"
```

## Что показывают трейсы

Каждый пример содержит:
1. Исходный вопрос пользователя
2. Полный ReAct-трейс (Thought → Action → Observation → ... → Answer)
3. Итоговый ответ агента
4. Комментарии по тому, какие инструменты были вызваны и почему

> Цифры в трейсах зависят от актуального курса валют на момент прогона.
