# Антигаллюцинация: поведение агента при ошибке API

## Сценарий

Оба источника курсов (frankfurter.app и ЦБ РФ) недоступны.
Агент не должен выдумывать курс.

## Ожидаемый ReAct-трейс

```
============================================================
[Thought]
Пользователь спрашивает "Сколько я потрачу в ближайшие 30 дней в рублях?".
Нужны обязательства и конвертация в RUB.
============================================================

============================================================
[Action]
tool=get_obligations
input={"status": "active", "category": null}
============================================================

============================================================
[Observation]
[11 записей]
============================================================

============================================================
[Action]
tool=convert_currency
input={"amount": 9.99, "from_currency": "USD", "to_currency": "RUB"}
============================================================

============================================================
[Observation]
{"ok": false, "error": "all currency providers failed: frankfurter:
 HTTPStatusError: 404 Not Found | cbr: ConnectError: Network unreachable"}
============================================================

============================================================
[Thought]
frankfurter.app и ЦБ РФ оба недоступны. Я НЕ буду выдумывать курс.
Сообщу пользователю, какие суммы есть в исходной валюте, и что
конвертация не удалась.
============================================================

--- ANSWER ---
Не удалось получить актуальный курс валют: оба источника недоступны.

- frankfurter.app: HTTP 404 (или сетевая ошибка)
- ЦБ РФ: Network unreachable

Я не могу выдумывать курсы, поэтому приведу исходные суммы:

В ближайшие 30 дней (2026-07-15 .. 2026-08-14) у вас списания:
- Netflix: 9.99 USD
- Spotify: 5.99 USD
- Яндекс.Плюс: 299.00 RUB (без конвертации)
- iCloud+: 2.99 USD
- Мосэнерго: 1500.00 RUB
- Ростелеком: 800.00 RUB
- Patreon: 5.00 EUR
- World Class: 4500.00 RUB
- The Guardian: 5.99 GBP
- JetBrains: 24.90 EUR
- GitHub Pro: 4.00 USD

Сумма в RUB: 7100.00 RUB (только исходные RUB-записи).
Конвертация USD/EUR/GBP не выполнена из-за ошибки API.

Рекомендация: попробуйте позже или используйте --force-refresh.
```

## Что демонстрирует

1. **Антигаллюцинация** — ключевое требование ТЗ.
2. **Структурированная ошибка** — `{"ok": false, "error": "..."}`.
3. **Частичный ответ** — что можно посчитать без API, агент считает.
4. **Рекомендация** — что делать пользователю дальше.
