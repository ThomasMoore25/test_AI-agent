# Contributor Covenant Code of Conduct

## Our Pledge

Мы стремимся создать открытую и гостеприимную среду для всех участников.

## Our Standards

**Позитивное поведение:**
- Уважительное отношение к разным точкам зрения
- Конструктивная критика
- Эмпатия к другим участникам сообщества

**Недопустимое поведение:**
- Оскорбления, унижения, троллинг
- Публикация личных данных без согласия
- Любые формы дискриминации

## Enforcement Responsibilities

Maintainer (ThomasMoore25) отвечает за соблюдение стандартов и может
удалять комментарии/коммиты, нарушающие данный Code of Conduct.

## Scope

Действует во всех пространствах проекта: issues, PR, commits, discussions.

## Enforcement

Нарушения сообщайте на ThomasMoore25@users.noreply.github.com.
